# Vendor_Summary_Analysis
An end-to-end EDA on a liquor inventory database with 12M+ sales records. Analyzes vendor profitability, brand pricing, bulk purchasing savings, and unsold inventory across the full supply chain — from purchases and invoicing to final sales — using Python, Pandas, SQLite, and SciPy.
